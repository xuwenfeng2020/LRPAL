# LRPAL
This is the demo code for the LRPAL framework

The public dataset can be downloaded from the website at the following link: https://rose1.ntu.edu.sg/dataset/actionRecognition/

Please process the data according to the data preprocessing methods described in the paper; the self-built dataset is not provided at this time due to its involvement in a cross-border project and the need for confidentiality.



## System requirement

#### Programming language
```
Python 3.8.20
```
#### Python Packages
```
PyTorch 2.1.0
CUDA 11.8.1
torch-geometric 2.2.0
numpy   1.24.4
scikit-learn 1.3.2
scipy   1.10.1
```

## Run the demo code

For the NTU RGB+D120 dataset, run with defualt setting
```
python3 main.py

```
